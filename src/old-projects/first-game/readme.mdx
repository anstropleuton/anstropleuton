# First Experiment

This is the final version of my very first experiment with [raylib](https://raylib.com). I created it when I was 16 years old using my Dad's laptop.

It started as just a simple test to render a square, resize it, rotate it and move it, but evolved with experiments using `Camera2D`, loading textures, position and velocity tracking, some Hue-shifting coloring (manual) and even debug info on screen.

It is not really a "game", as it does not have any objectives or even any other entity other than a simple texture.

## How To Run

Run the following command to compile and run the "game":

```
git clone https://github.com/anstropleuton/anstropleuton.git
cd anstropleuton/src/old-projects/first-game
make
./first-game
```
